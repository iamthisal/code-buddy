Coder Buddy is an agentic AI system that turns a single natural-language prompt into a fully working web application. It uses a LangGraph pipeline of three cooperating agents — a planner that breaks the request into features and files, an architect that expands each file into detailed implementation steps, and a coder that writes the actual code using tool-calling (via create_react_agent) to read and write files on disk. Powered by Groq's open-source LLMs via LangChain.
